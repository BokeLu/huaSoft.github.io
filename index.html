    # 关于内容差异说明

    本文为“华软智云管理系统”通用版本的用户使用手册，手册中出现的截图为演示系统的截图，并不代表真实用户数据。 本系统的名称、应用程序的发布名称、系统Logo、界面风格、菜单、目录结构、用户角色等能够根据客户的需求进行修改。功能方便的需求也能够根据客户的实际需求进行深度定制，因此，本手册所提及的操作、图标及名称可能与用户实际使用的产品会有所差异。

    # 本文符号约定

    为了便于表达，本文使用了以下字符:

    字 符 含 义
    【】 表示按钮或操作，例如【上传】
    < > 表示菜单，例如<系统管理>表示系统管理菜单
    [ ] 表示目录结构，例如[项目库]表示项目库目录
    - 菜单层次分隔符，例如<系统管理-角色管理 >表示“系统管理”菜单下的“角色管理”菜单
    / 目录层次分隔符，例如[项目库/电子商城股权投资项目/投前资料]表示“项目库”目录下的“电子商城股权投资项目”下的“投前资料”目录

    # 各类图标含义

    警告：提醒操作要非常小心，操作不当，将有可能造成数据丢失，系统崩溃。

    小心、注意：提醒注意事项，忽略这些事项，可能会影响到系统的稳定工作。

    说明，提示，技巧：对操作内容进行一些补充和说明，遵循这些补充、说明、提示，操作将更加简便。
